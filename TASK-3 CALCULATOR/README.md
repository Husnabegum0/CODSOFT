# SmartCalc

A minimal, responsive calculator with a soft pastel gradient background. Built with pure HTML, CSS, and JavaScript — no dependencies, no build step.

## Preview

> Pastel rose → lavender → mint → butter gradient background with a frosted glass card UI.

## Features

- Soft animated pastel gradient background
- Frosted glass calculator card
- Full arithmetic — `+` `−` `×` `÷`
- Utility — toggle sign (`+/−`), percentage (`%`)
- Keyboard support
- Division by zero error handling
- Responsive for mobile & desktop

## File Structure

```
smartcalc/
├── index.html   # Markup
├── style.css    # Pastel styles & layout
└── script.js    # Calculator logic
```

## Run Locally

Just open `index.html` in any browser — no server needed.

```bash
open index.html
# or
python -m http.server 8080
```

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch** → `main` / `(root)`
4. Click **Save**
5. Live at: `https://<username>.github.io/<repo>/`

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `0–9`, `.` | Input digit |
| `+` `-` `*` `/` | Operator |
| `Enter` or `=` | Equals |
| `Backspace` | Delete last digit |
| `Escape` / `Delete` | Clear all |
| `%` | Percentage |
